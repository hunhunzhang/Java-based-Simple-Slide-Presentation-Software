#java版本21.0.4
#IDE:IntelliJ IDEA Community Edition 2024.2.1
#源代码的编辑、编译、运行、测试都在此IDE中完成
在IDEA中新建java项目在将所有源代码文件复制粘贴到项目文件夹src中编译运行即可

也可通过命令行运行可执行文件运行程序：
			1.先进入可执行字节码文件目录
			2.运行Main.class: 
			如：D:\>cd "D:\桌面\可执行字节码"
				D:\桌面\可执行字节码> java Main

简单测试用VS直接打开源代码文件Main.java直接编译运行也行，不过会在文件夹内生成可执行字节代码
